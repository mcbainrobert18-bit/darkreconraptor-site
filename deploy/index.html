Njalla DNS → VPS Deploy

Action required: www.darkreconraptor.com currently CNAMEs to NinjaTech. Delete that record in Njalla and add an A record to your VPS IP.

1. Njalla A records

TypeHostValue
A@YOUR_SERVER_IP
AwwwYOUR_SERVER_IP
AhudYOUR_SERVER_IP
AbeastYOUR_SERVER_IP
AoperatorYOUR_SERVER_IP

2. One-click deploy (Windows)

cd C:\Users\xxdae\darkreconraptor_site\deploy
.\GO-LIVE.ps1 -ServerIp "YOUR_SERVER_IP"

Or double-click LINK-DOMAIN.bat in the site folder.

3. Nginx + SSL (VPS)

sudo cp /var/www/darkreconraptor/deploy/nginx.conf /etc/nginx/sites-available/darkreconraptor
sudo ln -sf /etc/nginx/sites-available/darkreconraptor /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
sudo certbot --nginx -d darkreconraptor.com -d www.darkreconraptor.com \
  -d hud.darkreconraptor.com -d beast.darkreconraptor.com -d operator.darkreconraptor.com
