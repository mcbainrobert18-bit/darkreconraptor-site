⚙️ KEEPING AN APP PERMANENTLY ALIVE ON YOUR VPS

Once your code is on a VPS, use systemd to make it restart automatically — even after reboots or crashes:

# 1. Create service file
sudo nano /etc/systemd/system/darkcakes.service

# Paste this:
[Unit]
Description=Dark Cakes POS
After=network.target

[Service]
WorkingDirectory=/home/user/cupcake_app
ExecStart=/usr/bin/python3 app.py
Restart=always
RestartSec=5
User=www-data

[Install]
WantedBy=multi-user.target

# 2. Enable and start
sudo systemctl daemon-reload
sudo systemctl enable darkcakes
sudo systemctl start darkcakes

# 3. Check status
sudo systemctl status darkcakes

That's it. Your app never dies. Reboots? Auto-starts. Crashes? Auto-restarts in 5 seconds.
